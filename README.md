# -CPU-Scheduling-Algorithm-in-Operating-Systems-
Objective Design and develop a program that simulates fundamental CPU Scheduling Algorithms, demonstrating your understanding of process management concepts in Operating Systems.

How to Use the Program

Enter the number of processes - minimum of 3. Click "Generate Inputs" to create the input fields. Fill in the following for each process:

Process ID (e.g., P1, P2, P3) Arrival Time Burst Time Priority (lower number = higher priority)

Enter the Time Quantum if using Round Robin or Priority with Round Robin. Select a scheduling algorithm from the dropdown:

First-Come, First-Served (FCFS) Shortest Job First (SJF) - Non-preemptive Shortest Remaining Time (SRT) - Preemptive Round Robin (RR) Priority Scheduling - Non-preemptive Priority Scheduling with Round Robin

Click "Run Simulation". View the results:

Gantt Chart - visual execution timeline Table — Waiting Time (WT) and Turnaround Time (TAT) per process Averages — Average WT and Average TAT
